= Security domains
:page-sidebar: am_3_x_sidebar
:page-permalink: am/current/am_userguide_security_domains.html
:page-folder: am/user-guide
:page-layout: am

A _security domain_ gives you access to all the Gravitee.io AM resources such as applications, users, identity providers, etc ...
It exposes authentication and authorization URLs and provides analytics and reporting.
